# Nomi Pizza - React Challenge

This is a React-based web application built with Vite for the "Pizzeria Mamma Mia" challenge. The project simulates a pizzeria website, allowing users to explore a menu, add items to a cart, and enjoy a smooth, responsive experience. It showcases modern React practices, state management, and a clean UI.

## Features

- **Menu Display**: Browse a variety of pizzas with details like name, price, and ingredients.
- **Shopping Cart**: Add pizzas to your cart and manage your order seamlessly.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Fast Development**: Built with Vite for a lightning-fast development experience and optimized production build.

## Tech Stack

- **React**: JavaScript library for building user interfaces.
- **Vite**: Next-generation frontend tooling for fast builds and hot module replacement.
- **CSS**: Custom styles for a polished and intuitive design.
- **JavaScript (ES6+)**: Modern JavaScript features for efficient coding.

## Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/) as a package manager

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/jpberse/react-desafio-pizzeria-mamma-mia.git

2. Navigate to the project folder:
   ```bash
   cd react-desafio-pizzeria-mamma-mia

3. Install dependencies:
   ```bash
   npm install

### Running the Project
- Start the development server:
  ```bash
  npm run dev

- Build for production:
    ```bash
  npm run build

- Preview the production build locally:
  ```bash
  npm run preview
